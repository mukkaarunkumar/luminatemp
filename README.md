LuminaTemp - Experience Climate

## Run Locally
Go to frontend folder
**Prerequisites:**  Node.js, Razorpay API KEYS

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev` or `parcel index.html`

Go to backend folder
**Prerequisites:**  Mongo DB, Razorpay API KEYS, Firebase
1. Install dependencies:
   `npm install`
2. Run the app:
   `node server.js`
